                         Project Development Objective (from Project Appraisal Document)

                         Improve the security and reliability of the electricity transmission system and wholesale electricity supply and, therefore, facilitate unimpeded commercial

                         Implementation Status Overview
                         The project had achieved the original key performance indicators by the end of 2009. The electrical component has been implemented with the exception of the Telecommunication
                         and SCADA contract, which is still experiencing delays. The implementation of the additional financing project is in an advanced stage. The Heating Component contracts are
                         expected to be completed by the Project Closing Date on December 31, 2011.
