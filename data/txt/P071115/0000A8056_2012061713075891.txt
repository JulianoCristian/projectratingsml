                         Project Development Objectives
                         Project Development Objective (from Project Appraisal Document)

                         The Project builds upon and anchors more deeply the institutional reforms started under the first Export Development project with the objective of creating a conducive
                         export environbment encouraging trade. Its development objectives are to improve access to export markets and finance, and enhance the efficiency trade logistics.
                         The Project is expected to create linkages between domestic and off-shore sectors, enhance competitiveness of private enterprises, and strengthen public-private
                         interface to administer and promote internal and external trade.

                         Implementation Status Overview
                         Overall, the implementation of the project is making good progress. The project has met most of its targets. The project closing date has been extended from March 31, 2012 to
                         September 30, 2012 to finalize some tasks and ensure a proper follow up as well as ensuring a smoother transition to a possible EDP III. These tasks include:
Public Disclosure Copy




                         (i) Finalizing the last batch of FAMEX applications export plans
                         (iii) Finalizing acquisition and installation of equipments for the Customs Administration
                         (iii) Installing and testing equipment for the technical control component
                         (iv)Finalizing application for the export guarantee component ( Dahmen Finance)
