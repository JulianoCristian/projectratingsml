                         Key Decisions Regarding Implementation
                         The loan has been extended to June 15, 2011. It is expected that all activities except SCLR will be completed by that time. The completion of SCLR in a time bound manner by
Public Disclosure Copy




                         December 2011 remains to be confirmed. A set of nine key actions has been agreed that should allow MMRDA to submit to the Bank, by December 2010, a revised time bound
                         implementation plan to complete successfully the SCLR and the rest of the project. Should this implementation plan be found satisfactory to the Bank the implementation progress of
                         the project would be rated satisfactory and a further extension to enable successful completion of the project would be considered.
